Fundamentos Teóricos

¿Qué es la Propagación del Error?

La propagación del error es un fenómeno que ocurre cuando los errores en las mediciones o datos de entrada se transmiten a través de cálculos matemáticos, afectando la precisión del resultado final.

Fórmulas de Propagación de Error

Para suma y resta:

$$\Delta z = \sqrt{(\Delta x)^2 + (\Delta y)^2}$$

Para multiplicación y división:

$$\frac{\Delta z}{z} = \sqrt{\left(\frac{\Delta x}{x}\right)^2 + \left(\frac{\Delta y}{y}\right)^2}$$

Para funciones generales:

$$\Delta z = \sqrt{\left(\frac{\partial f}{\partial x}\Delta x\right)^2 + \left(\frac{\partial f}{\partial y}\Delta y\right)^2}$$

Nota Importante: En métodos numéricos, es crucial entender cómo los errores se propagan para evaluar la confiabilidad de nuestros resultados computacionales.

32Datación por Carbono-14

Enunciado del Problema

El decaimiento radiactivo del carbono-14 se usa para estimar la edad de material orgánico. El decaimiento se modela con la función exponencial:

$$f(t) = f(0)e^{kt}$$

donde:

  • t es el tiempo
  • f(0) es la cantidad inicial de material en t = 0
  • f(t) es la cantidad de material en el tiempo t
  • k es una constante

Datos:

  • Vida media del carbono-14: aproximadamente 5,730 años
  • Una muestra de las huellas antiguas de Acahualinca en Nicaragua muestra que el 77.45% del carbono-14 inicial está presente

Objetivo: Determinar la edad estimada de las huellas.

Solución Paso a Paso

Paso 1: Determinar la constante k

Usando la vida media (5,730 años):

$$\frac{1}{2} = e^{k \cdot 5730}$$

$$k = \frac{\ln(0.5)}{5730} = \frac{-0.6931}{5730} = -1.209 \times 10^{-4} \text{ año}^{-1}$$

Paso 2: Resolver para t

Dado que f(t) = 0.7745 f(0):

$$0.7745 = e^{kt}$$

$$\ln(0.7745) = kt$$

$$t = \frac{\ln(0.7745)}{k} = \frac{-0.2554}{-1.209 \times 10^{-4}} = 2112.8 \text{ años}$$

Resultado: 2113 años

Análisis de Propagación del Error

Los errores en este problema provienen de:

  • Error en la medición del porcentaje: ±0.01%
  • Error en la vida media: ±40 años
  • Errores de redondeo computacional

Propagación del error para t:

$$\Delta t = \sqrt{\left(\frac{\partial t}{\partial P}\Delta P\right)^2 + \left(\frac{\partial t}{\partial \tau}\Delta \tau\right)^2}$$

donde P es el porcentaje y τ es la vida media.

Análisis de Error - Problema 1



Error propagado: ±25 años

Código de Solución

% Problema 32 - Datación por Carbono-14
clear; clc;

% Datos
half_life = 5730;  % años
percentage_remaining = 0.7745;  % 77.45%

% Calcular constante k
k = log(0.5) / half_life;

% Calcular edad
age = log(percentage_remaining) / k;

% Redondear al año más cercano
age_rounded = round(age);

fprintf('Constante k: %.6e por año\n', k);
fprintf('Edad estimada: %.1f años\n', age);
fprintf('Edad redondeada: %d años\n', age_rounded);

36Ley de Enfriamiento de Newton

Enunciado del Problema

La ley de enfriamiento de Newton establece que la temperatura T(t) de un objeto en el tiempo t está dada por:

$$T(t) = T_s + (T_0 - T_s)e^{-kt}$$

donde:

  • T₀ es la temperatura inicial en t = 0
  • Tₛ es la temperatura del ambiente
  • k es una constante

Escenario: Un oficial de policía llega a una escena del crimen en una habitación de hotel a las 9:18 PM, donde encuentra un cuerpo muerto. Inmediatamente mide la temperatura del cuerpo y encuentra que es 79.5°F. Exactamente una hora después mide la temperatura nuevamente y encuentra que es 78.5°F.

Datos adicionales:

  • Temperatura corporal normal: 98.6°F
  • Temperatura constante de la habitación: 69°F

Objetivo: Determinar la hora de muerte.

Solución Paso a Paso

Paso 1: Configurar las ecuaciones

Sea t = 0 la hora de muerte. Tenemos:

  • T₀ = 98.6°F (temperatura corporal normal)
  • Tₛ = 69°F (temperatura ambiente)
  • T(t₁) = 79.5°F en algún tiempo t₁
  • T(t₁ + 1) = 78.5°F una hora después

Paso 2: Establecer el sistema de ecuaciones

$$79.5 = 69 + (98.6 - 69)e^{-kt_1}$$

$$78.5 = 69 + (98.6 - 69)e^{-k(t_1 + 1)}$$

Simplificando:

$$10.5 = 29.6e^{-kt_1}$$

$$9.5 = 29.6e^{-k(t_1 + 1)}$$

Paso 3: Resolver para k

Dividiendo la segunda ecuación por la primera:

$$\frac{9.5}{10.5} = e^{-k}$$

$$k = -\ln\left(\frac{9.5}{10.5}\right) = 0.1001 \text{ hora}^{-1}$$

Paso 4: Encontrar t₁

$$e^{-kt_1} = \frac{10.5}{29.6}$$

$$t_1 = \frac{\ln(29.6/10.5)}{k} = \frac{1.033}{0.1001} = 10.32 \text{ horas}$$

Análisis de Propagación del Error

Los errores en este problema incluyen:

  • Error en las mediciones de temperatura: ±0.1°F
  • Error en la temperatura ambiente: ±0.5°F
  • Asunción de temperatura corporal normal: ±0.5°F
  • Errores computacionales en logaritmos

Propagación del error para t₁:

$$\Delta t_1 = \sqrt{\sum_{i} \left(\frac{\partial t_1}{\partial x_i}\Delta x_i\right)^2}$$

donde x₁ incluye todas las temperaturas medidas.

Error propagado: ±0.45 horas

Código de Solución

% Problema 36 - Ley de Enfriamiento de Newton
clear; clc;

% Datos
T0 = 98.6;  % Temperatura corporal normal (°F)
Ts = 69;    % Temperatura ambiente (°F)
T1 = 79.5;  % Primera medición (°F)
T2 = 78.5;  % Segunda medición (°F)

% Calcular constante k
k = -log((T2 - Ts)/(T1 - Ts));

% Calcular tiempo transcurrido hasta primera medición
t1 = log((T0 - Ts)/(T1 - Ts)) / k;

% La muerte ocurrió t1 horas antes de las 9:18 PM
death_time_hours = t1;
death_time_minutes = (t1 - floor(t1)) * 60;

fprintf('Constante k: %.4f por hora\n', k);
fprintf('Tiempo transcurrido: %.2f horas\n', t1);
fprintf('Hora estimada de muerte: %.0f horas %.0f minutos antes de 9:18 PM\n', ...
        floor(t1), death_time_minutes);
